# AI Project Framework

Este directorio contiene una metodología simple para desarrollar proyectos de Data Analytics asistidos por IA.

No pretende reemplazar metodologías tradicionales (CRISP-DM, TDSP o Agile), sino complementarlas con buenas prácticas para el uso profesional de modelos de IA.

El framework se basa en cinco pilares:

1. Metodología de desarrollo.
2. Buenas prácticas de Prompt Engineering.
3. Revisión técnica del proyecto.
4. Auditoría y gobernanza de IA.
5. Checklist de calidad.

Puede reutilizarse en cualquier proyecto de SQL, Power BI, Python, Machine Learning o Business Intelligence.
